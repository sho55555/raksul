# アプリケーションを作成した背景

シフト制の職場において「締め切り日までにシフトが集まらない」という課題を抱えていた。課題を分析した結果、「シフトの提出場所が職場に限られる」ことや、「シフトの提出自体を忘れてしまう」ことが真因であると仮説を立てた。同様の問題を抱えている方も多いと推測し、真因を解決するために、場所に縛られずシフトの提出・確認・作成ができるWEBアプリケーションを開発することにした。
